# Focus

Adaptive agent enhancement distilled from 8 AI agent frameworks.

**253 lines. 0 dependencies. 3 files.**

## What It Does

1. **Context persistence** — Hooks re-inject your plan before every tool call. Never loses track.
2. **Adaptive process** — Typo fix = zero ceremony. Database redesign = real plan with human review.
3. **Cross-session memory** — Remembers decisions, patterns, and open items across sessions.
4. **Failure handling** — Log before retry. Never repeat same approach. Three strikes = ask human.
5. **Human steering** — Proceed on silence. Ask only when genuinely uncertain.

## How It Works

| Level | Example | Process |
|-------|---------|---------|
| **Trivial** | Fix typo | Just do it. |
| **Small** | Add a function | 3-line plan in log. Do it. |
| **Medium** | New API endpoint | Plan with 2-4 phases. Start working. |
| **Large** | Auth redesign | Research → spec → ask human → execute. |

Tasks escalate mid-work if they turn out bigger than expected.
